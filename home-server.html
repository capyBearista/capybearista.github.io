Hardware I'm Using

The hardware setup is intentionally modest and cost-effective, demonstrating that powerful home server capabilities don't require expensive enterprise equipment:

  • 2020 Intel MacBook Pro, 13-inch - Repurposed laptop serving as the main server, providing pretty reliable performance with low power consumption

  • Tenda SG105 Switch - 5-port gigabit switch

  • 2TB Seagate Portable HDD - External storage

  • A bunch of Cat6 RJ45 cables - Cables. I love cables.

Software I'm Using

  • Ubuntu Server - My chosen Linux distribution for this project, providing a stable foundation

  • Docker & Docker Compose - Containerization platform that makes deploying and managing applications incredibly simple and consistent

  • Pi-hole - Network-wide ad blocker that filters DNS requests, protecting all devices on the network from ads and trackers

  • Jellyfin - Self-hosted media streaming service that transforms the server into basically a personal Netflix for movies, TV shows, and music

  • Immich - Google Photos alternative for backing up and organizing family photos with AI-powered features like facial recognition

  • Nextcloud - Complete file hosting and collaboration platform for secure cloud storage and document sharing

  • Minecraft & Valheim Servers - Game servers for friends and family to enjoy multiplayer gaming together :)

  • DuckDNS - Dynamic DNS service that provides stable remote access even with changing IP addresses, because Xfinity sucks

  • Portainer - Web-based Docker management interface for easy container administration
